Lasagne recipe

Description

This is a easy and quick recipe, to make a basic lasagne. It takes a while because it has to cook for some time.

Ingrediense

  • Meat
  • Pasta plates
  • sauce
  • Cheese

Steps

  1. Cook the meat
  2. Add sauce to the meat
  3. to layer by layer, start with pasta plate
  4. Add cheese to the top
  5. Cook in the oven for 1 hour
